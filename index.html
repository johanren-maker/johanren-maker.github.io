📤 Export Database

Download a full backup of all months, readings and billing data as a JSON file. Use this to transfer data between browsers or computers.

📥 Import Database

Restore from a previously exported JSON backup. This will merge with existing data — existing months will be overwritten if the same key exists.

Export all months, readings and recovery analysis to a formatted Excel workbook with one sheet per month per utility.
