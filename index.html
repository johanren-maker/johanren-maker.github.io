Database Management

📤 Export Database

Download a complete backup of all data — months, readings, billing, unit owners, property settings and user accounts — as a single JSON file.

📥 Import Database

Restore from a previously exported backup JSON. Restores months, unit owners and property settings. You will be asked separately whether to restore user accounts.

📊 Export to Excel

Export all months, readings and recovery analysis to a formatted Excel workbook with one sheet per month per utility.

🐙 GitHub Backup

Automatically commit a full database backup to a GitHub repository on a schedule or on demand. Requires a GitHub Personal Access Token with repo scope.

🎨 Appearance

Customize the look and feel of the application.

⚠️ Danger Zone

Permanently delete all months, readings, billing data and settings from the local database. This action cannot be undone. Export a backup first.
